# 高校学生信息管理系统 python812

## 项目概述

高校学生信息管理系统是一个基于Python开发的，旨在帮助高校更高效地管理和处理学生信息的数据驱动平台。

## 技术栈

- 后端：Python 3.x，Flask框架
- 数据库：MySQL
- 前端：HTML, CSS, JavaScript

## 功能模块

- **学生信息管理**：包括学生信息的增加、删除、修改和查询功能。
- **成绩管理**：支持成绩录入、修改和查询。
- **课程管理**：包括课程的添加、删除和修改。
- **用户权限管理**：区分管理员和普通用户，实现不同权限的访问控制。

## 系统角色

- 系统管理员：负责系统的维护和用户管理。
- 教师：负责录入和修改学生成绩，管理课程信息。
- 学生：查询个人信息和成绩。

## 运行环境

- 操作系统：兼容Linux和Windows。
- Python版本：Python 3.x。
- 依赖管理：使用pip管理Python依赖。

## 部署步骤

1. 安装Python环境。
2. 使用pip安装所需依赖。
3. 配置数据库MySQL，并导入初始数据。
4. 部署前端静态文件。
5. 运行Flask应用。

## 目录结构

```
.
├── app               # Flask应用目录
│   ├── __init__.py
│   ├── models.py     # 数据库模型
│   ├── views.py      # 视图层逻辑
│   └── templates     # HTML模板目录
├── static            # 静态文件目录
├── venv              # 虚拟环境目录
└── requirements.txt  # 依赖列表
```

## 核心亮点

- **数据安全**：通过用户权限管理确保数据的安全性。
- **灵活扩展**：基于模块化的设计，易于扩展新的功能。
